# SCI

NOTE: This is a project in progress. 
This gem will calculate a sequencing complexity index for BAM files.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'SCI'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install SCI --pre
