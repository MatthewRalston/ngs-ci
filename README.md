# LCI

NOTE: This is a project in progress. 
This gem will calculate a library complexity index for BAM files.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'LCI'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install LCI
